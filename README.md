🤖 AI Life Decision Simulator

A domain-specific generative AI chatbot that helps users make 
structured, informed life decisions using Google Gemini API.

Built with Python Flask + HTML/CSS/JS frontend.

Features:
- 6 categories: Career, Money, Relationship, Business, Lifestyle, Investment
- Smart responses based on question complexity
- Clean dark-themed chat UI
- Powered by Google Gemini 2.5 Flash

Tech Stack: Python | Flask | Google Gemini API | HTML | CSS | JavaScript
